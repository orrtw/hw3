>Исследуем ряд exp(x)
```html
<x=1
eps=10**(-6) 
s=x 
t=x 
i=1
while abs(t/s)>eps: 
	t=(x/i)*t
	s=s+t 
	i=i+1 
import math
a=math.fabs(s-math.exp(x))/math.exp(x)
 
print ('s=',s, 'tmax=',t, 'a=',a, 'i=', i)>
```

>Используем "плохую" версию алгоритма.
```html
<import math
x=1
eps=10**(-5) 
s=x 
t=x 
i=1
while abs(t/s)>eps: 
	import math
	t=x**i/math.factorial(i)
	s=s+t 
	i=i+1 
import math
a=math.fabs(s-math.exp(x))/math.exp(x)
print ('s=',s, 'tmax=',t, 'a=',a, 'i=', i)
>
```
